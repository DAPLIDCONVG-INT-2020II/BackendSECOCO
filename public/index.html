SECOCO

Una app para los tiempos de pandemia

Colaboradores Brandon R, Diego M, Cristian C

Descripción

Registrate, permitenos saber tu ubicación, reporta tus sintomas y si has tenido contacto con alguien con COVID-19 te notificaremos. Tambien podras ver el estado de tu localidad por si deseas tomar una cuarentena preventiva o tener más precaución.

Nota: Necesitas al menos un 25% de bateria para que puedas utilizar adecuadamente las funciones del app.

Información tecnica

Este es un servidor con servicios API REST, desplegado en Glitch y elaborado en Node.js utilizando Express como framework, mientras la capa de persistencia pertenece a Google Cloud Firestore.
